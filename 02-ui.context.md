# UI / UX Design

Theme:

Modern Hospital Dashboard

Design Inspiration:

Train seat reservation systems.

The bed map should be the hero section of the dashboard.

Use a blue and white color scheme.

Design Principles:

- Clean
- Minimal
- Professional
- Calm
- Easy to scan quickly
- Suitable for nurse stations

Color Palette:

Primary:
#2563EB

Background:
#F8FAFC

Surface:
#FFFFFF

Border:
#E5E7EB

Bed States:

NORMAL:
Blue

ALERT:
Red

OFFLINE:
Gray

Typography:

Font:
Inter

Rounded Corners:
16px

Soft Shadows

Header

B1 B2 B3

B4 B5 B6

- Bed Number
- Device Status
- Patient ID
- Patient Name
