# Project Overview

Project Name:

Patient Bed Exit Alert System

Description:

A real-time hospital dashboard that alerts nurses when patients leave their beds without assistance.

The system monitors 6 hospital beds using ESP32 devices.

The purpose of this system is to prevent patient falls and notify medical staff immediately when a patient gets out of bed.

Target Users:

- Nurses
- Healthcare Staff

Supported Beds:

6 Beds

Page Type:

Single Page Dashboard

Core Features:

- Monitor device online/offline status.
- Detect when a patient leaves the bed.
- Show active alerts immediately.
- Maintain alert history.
- Display a visual bed map.
